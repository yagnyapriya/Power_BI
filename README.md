# IBM HR Attrition Analysis using Power BI
## Description
An HR Attrition Analysis built using Microsoft Power BI on the IBM HR dataset. The project highlights key attrition trends across tenure, roles, departments, salary, and satisfaction to support data-driven HR insights and retention decisions.
